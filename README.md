# obtain-fqdn-during-tls-using-aws-nw-fw

## Description

You can easily create validation resources to obtain the FQDN during TLS communication using AWS Network Firewall with Terraform.

## Requirements

- Terraform version is ">= 1.4.0"

## Using

### Create resources

```sh
terraform init
terraform apply
```

### Delete resources

```sh
terraform destroy
```
